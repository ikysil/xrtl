XRTL Readme
---------------------------------------

1. What is XRTL?
XRTL - eXtended Run-Time Library for Delphi.
XRTL contains:
- input-output classes (xrtl_io)
- compress/decompress support based on zlib and bzlib
  (xrtl_io_zlib and xrtl_io_bzlib)
- cryptographic primitives (xrtl_crypt, xrtl_crypt_cipher, xrtl_crypt_md)
- arbitrary precision math support (xrtl_math)
- OPC server SDK (xrtl_opc_sdk)
- misc. utils (xrtl_util)
